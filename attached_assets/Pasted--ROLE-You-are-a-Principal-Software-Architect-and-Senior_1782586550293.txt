Tagline:

# PROJECT DESCRIPTION

Build a complete AI-powered tutoring platform using Retrieval-Augmented Generation (RAG).

Traditional learning platforms provide static information.

ClarifAI should simulate a personalized AI tutor that teaches students using ONLY the uploaded educational material instead of hallucinating answers.

The platform must support document uploads, semantic search, conversational tutoring, adaptive quizzes, flashcards, analytics, and learning progress.

The AI must NEVER answer outside the uploaded documents.

If the answer cannot be found, it should politely state that the information is unavailable in the uploaded study material.

---

# PRIMARY GOAL

Develop a modern AI tutoring platform capable of

• Uploading educational PDFs, DOCX and TXT files

• Converting them into embeddings

• Storing embeddings inside ChromaDB

• Performing semantic search

• Retrieving relevant context

• Generating accurate responses using a local LLM

• Maintaining conversational memory

• Generating quizzes

• Creating flashcards

• Providing analytics for educators
